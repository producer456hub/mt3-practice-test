# Biol 40a — Midterm 3 Practice Test

A 150-question multiple-choice practice test for Biology 40a (Human Anatomy & Physiology) Midterm 3,
covering **Bones (60)**, **Muscles (60)**, and **Joints (30)**. Built from the Week 7–11 lecture
decks, the LA7–LA11 lab activities, and the in-class station activities.

## Features
- 6 answer choices per question (1 correct + 5 distractors)
- Shows the correct answer and a short explanation after every question
- **Mastery loop:** when the test ends, every question you missed comes back in repeat rounds until
  you've answered all of them correctly
- Pick topics, shuffle questions/choices, keyboard shortcuts (1–6, Enter/Space)

Open `index.html` (questions live in `questions.js`). Live on GitHub Pages.
